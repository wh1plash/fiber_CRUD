# CRUD API by using Fiber framework

## Start application
### first of all you must boot up yor DB and metrics instances.
```
> docker-compose up
```
### then run server
```
> make run
```

## Exanple of .env file
```
PG_HOST="localhost"
PG_PORT=5444
PG_USER="postgres"
PG_PASS="postgres"
PG_DB_NAME="Fiber_CRUD"
```
